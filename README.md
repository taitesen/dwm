# NOTE
---

This install script is composed of moving files around only. It was design for personal use in arch linux, and might not be suited for everyone. It is highly suggested to tweak as per your distribution/desktop enviroment needs.

Tih ve leh lawk loh chuan!!

- With wget
```sh
wget -c https://raw.githubusercontent.com/taitesen/dwm/master/install.sh
chmod +x install.sh
./install.sh
```
- With curl
```sh
curl -C - -O https://raw.githubusercontent.com/taitesen/dwm/master/install.sh
chmod +x install.sh
./install.sh
```
